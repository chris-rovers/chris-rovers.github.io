WEBSITES

Since finishing my program at triOS, I have made a website for Aeriform Aerial Analysis. It is a static site written in HTML, CSS (Flexbox and Bootstrap) and a little JavaScript. There is also a simple PHP mailer script for the contact form.

Below are various web projects that I have made during my time in the Web Developer program. Most courses in the program span four weeks. Typically, a course consists of about one to two weeks of instruction where we would start a basic project together as a class to get used to the new language or tool. The next two to three weeks are spent working on our individual projects with the knowledge we just learned. As you can imagine, this requires the ability to learn quickly and good research/problem-solving skills to add to the foundational knowledge provided during instruction time.
