WEBSITES

Below are various web projects that I have made over the past year and a half in my Web Developer program. Courses in the program span four weeks each. Typically, a course consists of about one to two weeks of instruction and starting something basic together as a class, with the given language/tool we are focusing on. The next two to three weeks are spent working on our individual projects with the knowledge we just learned. As you can imagine, this requires the ability to learn quickly and good research/problem-solving skills to add to the foundational knowledge provided during instruction time.

Cancer Bats - Oct. 2018

This is a website I made about the band Cancer Bats.

This was my first web project and uses only HTML, CSS and JavaScript.

It was a good learning experience in terms of floats and the importance of image sizing.

I included functionality that mimicked adding items to a cart and displaying a total.

Show Sherpa - Feb. 2019

This is my second web project.

It uses Razor Pages in ASP.NET Core.

It pulls data from The Movie Database API.

This was my first project using a REST API.

VisionQuest - May 2019

This was my third web project, written in PHP.

The project had to incorporate a store and shopping cart.

A partner and I worked together, using MySQL for the back-end database.

I designed the database and populated the tables myself.

NHL Player Stats - Jul. 2019

This was my fourth web project.

While fairly basic and admittedly plain looking, it was made in Angular so my main focus was on learning the framework, rather than design.

Being an Angular app, it was of course written in TypeScript.

I used the MySportsFeeds.com API for the data.

ComicsPy - Oct. 2019

My fifth web project was written in Python, using the Django web framework.

I used the ComicVine API for the data.

It is the first site I've made that primarily uses Bootstrap for the layout and styling.

I like the power and flexibility that frameworks provide.
